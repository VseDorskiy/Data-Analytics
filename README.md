- Analytical review of a global P-2-P lending portfolio (670k+ loans) sourced from the platform (www.kiva.org). 
The project examines geographical patterns, sector distributions, borrower characteristics, and lending dynamics across multiple countries.

- Data sourced from CSV files; processing, cleaning, and analysis performed in Python within Jupyter Notebook.

- Final results compiled into an interactive HTML report (Analyse_von_kiva.org.html).

Project structure:

.

├─ Projektdaten/                          # Source files and Jupyter Notebooks used to generate the report

├─ Aufgabe/                               # Original project task and instructions

└─ Analyse_von_kiva.org.html              # Final project report (rendered HTML)
